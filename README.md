# Manga Compactor

该脚本将原始的图片格式按照作者-作品-章节的树形结构打包成一个Epub文件。

## 使用说明

1. 图片必须按照如下目录格式放置，其中不要求作者和作品的名称格式，但是要求章节中带一个ascii阿拉伯数字作为排序，例如《第1话 AABBCC》；且章节内的图片要求包含一个“_p{n}”的部分作为识别图片顺序的依据，例如，“图片1_p1”或者“uuid_p11”。
2. 运行脚本应满足依赖，运行时要求传入一个参数，及工作目录的路径。输出的epub位置位于当前运行的目录而不是传入的工作目录。


```
{工作目录}
    ├──{作者A}
        ├── {作品AA}
            ├── {章节1}
                ├── xxx_p1
                ├── xxx_p2
                ├── ......
                └── xxx_pn
            ├── {章节2}
            ├── ......
            └── {章节n}
        ├── ......
        └── {作品NN}
    ├── ......
    └── {作者B}
```
